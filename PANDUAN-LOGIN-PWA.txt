================================================================================
                      TROUBLESHOOTING LOGIN
================================================================================

MASALAH: "Login via browser dinonaktifkan"
PENYEBAB: Anda membuka dari browser Chrome, bukan aplikasi PWA
SOLUSI:
- Tutup browser
- Buka aplikasi "Agunan" dari home screen
- Login dari aplikasi tersebut


MASALAH: "Username atau password salah"
SOLUSI:
- Pastikan username benar (huruf besar/kecil sesuai)
- Pastikan password benar (huruf besar/kecil sesuai)
- Cek apakah ada spasi di awal/akhir username/password
- Hubungi admin untuk reset password jika lupa


MASALAH: Tombol fingerprint tidak muncul
PENYEBAB: Belum aktivasi fingerprint atau buka dari browser
SOLUSI:
- Pastikan sudah login sekali dan aktivasi fingerprint
- Pastikan buka dari aplikasi PWA, bukan browser
- Logout dan login ulang untuk coba aktivasi lagi


MASALAH: Fingerprint gagal scan terus
SOLUSI:
- Pastikan jari bersih dan kering
- Pastikan sensor fingerprint HP bersih
- Coba gunakan jari lain yang sudah terdaftar di HP
- Jika masih gagal, gunakan login username/password


MASALAH: Setelah login, langsung logout lagi
PENYEBAB: Session terputus atau buka dari browser
SOLUSI:
- Pastikan buka dari aplikasi PWA di home screen
- Jangan buka dari browser biasa
- Clear cache aplikasi di Settings > Apps > Agunan
- Uninstall dan install ulang aplikasi


MASALAH: Aplikasi loading terus saat login
SOLUSI:
- Cek koneksi internet
- Pastikan server tidak down
- Restart aplikasi
- Restart HP jika perlu
